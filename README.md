# Digital Literacy Project

## Overview

This project is part of the CSE0001 Digital Literacy course. It aims to develop essential digital skills required for academic and professional growth. The project includes multiple tasks focused on digital awareness, online presence, communication, and cybersecurity.

## Objectives

* Understand the concept of digital literacy
* Learn safe and responsible internet practices
* Build a professional digital portfolio
* Explore coding and collaboration platforms
* Develop proper email communication skills
* Gain awareness about cybercrime and prevention

## Project Structure

* task-1-presentation
  Infographic on digital literacy awareness

* task-2-portfolio
  Screenshots of professional profiles (GitHub, LinkedIn, etc.)

* task-3-platforms
  Coding platform activity and Google Form quiz

* **task-4-email-etiquette/**
  Professional email drafts and social media checklist

* **task-5-cybercrime/**
  Cybercrime case study and prevention checklist

* report
  Final project report document

## Tools & Platforms Used

* Canva (Infographic design)
* GitHub (Project repository)
* LinkedIn (Professional profile)
* Google Forms (Quiz creation)
* Coding platforms (HackerRank)
